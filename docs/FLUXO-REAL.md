# Fluxo real confirmado

1. O morador entra por reconhecimento facial já existente.
2. O sistema cria uma sessão.
3. O morador circula pelo mercadinho.
4. Retira produtos.
5. Pode devolver produtos.
6. Registra produtos no autoatendimento.
7. Informa quantidades ou realiza vários bipes.
8. Realiza o pagamento.
9. Sai normalmente.

## Regra absoluta

A IA não bloqueia automaticamente a saída.

Mesmo quando existir possível divergência:

- a pessoa sai;
- a sessão é salva;
- eventos relevantes são preservados;
- uma ocorrência privada é criada;
- uma pessoa revisa antes de qualquer providência.

## Linguagem permitida

- possível divergência;
- evento para revisão;
- confiança estimada;
- imagem insuficiente;
- necessita revisão humana.

## Linguagem proibida automaticamente

- furto confirmado;
- ladrão;
- culpado;
- crime detectado;
- criminoso.
